
# StudyForge Roadmap

## Current: v1.7

Focus:
Building the foundation.

Features:
- AI tutor
- multiple subjects
- learning depth controls
- different learning modes
- personal chat history


## v1.8 — Personalisation

Goal:
Make StudyForge understand individual learners.

Planned:

- user accounts
- student profiles
- learning preferences
- personalised settings


## v1.9 — Learning Analytics

Goal:
Help students understand their own progress.

Features:

- weakness tracking
- topic mastery
- progress dashboard
- improvement suggestions


## v2 — Knowledge-Based Learning

Goal:
Make answers more reliable and personalised.

Features:

- upload notes
- textbook integration
- PDF support
- RAG-based answers


## v3 — Adaptive Learning

Goal:
Move from answering questions to actively helping students improve.

Features:

- AI study plans
- adaptive difficulty
- mistake analysis
- personalised revision
